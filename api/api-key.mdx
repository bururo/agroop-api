A **API Key** é a sua credencial única para aceder à Agroop API.  
Após ativar a subscrição, o nosso suporte criará a sua **entidade** e associará os dispositivos.

### Como obter a API Key:

1. Receberá um **email no endereço indicado** com um **link temporário**.
2. Clique no link e será redirecionado para a página da Agroop API.
3. Nessa página, verá a sua **API Key** – **copie e guarde num local seguro**.

> ⚠ O link expira após a primeira utilização ou se fechar a página.  
> Se perder a API Key, não se preocupe:

- No cabeçalho da API, clique em **"Recuperar API Key"**.
- Insira o **email da entidade**.
- Receberá um **novo link temporário** para recuperar a mesma chave.

_(Por motivos de segurança, não damos feedback imediato ao inserir o email – isto reduz ataques DDoS.)_  
Se tiver dificuldades, verifique a pasta **spam** ou contacte **support@agroop.net**.
