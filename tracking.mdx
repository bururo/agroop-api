A Agroop API foi desenhada para ser **rápida, segura e escalável**.  
Para garantir uma experiência equilibrada para todos os utilizadores, aplicamos boas práticas de controlo de tráfego.

### Limites de utilização:

- **Pedidos por minuto:** 60
- **Pedidos diários por entidade:** 10.000
- **Timeout por pedido:** 30 segundos

Embora os utilizadores não tenham acesso direto a estas métricas, monitorizamos cada pedido por entidade para manter a qualidade do serviço.

<Warning> Caso exceda estes limites, poderá receber erros.</Warning>
Em caso de dúvida, contacte-nos através de: **support@agroop.net**.
