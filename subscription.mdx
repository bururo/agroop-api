Para utilizar a **Agroop API**, é necessário ter uma **subscrição ativa**.  
A subscrição é a chave que permite à sua entidade aceder aos dados recolhidos pelos nossos dispositivos.

### Como funciona a subscrição?

- A subscrição é **associada à entidade** (empresa, ou organização, ou pessoa).
- Está vinculada a um endereço de e-mail, através do qual receberá o token aceder à API Key.
- A subscrição tem uma **duração mínima de 1 ano**, podendo ser renovada anualmente.
- A subscrição tem custos associados. Para mais informações, contacte: **support@agroop.net**.

<Warning>
  {' '}
  Sem uma subscrição ativa, não é possível gerar a API Key nem consultar dados. Os dados só estarão disponíveis durante o
  período de vigência da subscrição. Caso renove posteriormente e haja um intervalo entre as subscrições, os dados desse
  período não serão disponibilizados.
</Warning>
