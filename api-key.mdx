A **API Key** é a sua credencial única para fazer os pedidos na Agroop API.  
Para que possa utilizá-la, a nossa equipa de suporte terá de criar a sua entidade, atribuir uma subscrição e associar os dispositivos.

### Como obter a API Key:

1. Receberá um **email no endereço indicado** com um **link temporário**.
2. Clique no link e será redirecionado para a página da Agroop API.
3. Nessa página, verá a sua **API Key** – **copie e guarde num local seguro**.

<Warning> O link expira após a primeira utilização ou se fechar a página. </Warning>

<Tip>
  Caso perca a sua API Key, não se preocupe! No cabeçalho, clique em **"Recuperar API Key"** e introduza o email da
  entidade. Receberá um novo link temporário para recuperar a mesma chave.
</Tip>
<Warning>Por motivos de segurança, não damos feedback após introduzir o email. </Warning>

Se tiver dificuldades, verifique a pasta **spam** ou contacte-nos através de **support@agroop.net**.
